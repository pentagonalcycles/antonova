# TESoul'RA Website

## Local development
- `npm install`
- `npm run dev`

## Quality gates
- `npm run test:unit`
- `npm run test:e2e`
- `npm run build`

## Vercel deployment
1. Push commits to the working branch (preview deployments trigger automatically).
2. Configure environment variables from `.env.example` in Vercel project settings.
3. Verify the Vercel preview deployment for each pushed commit.
4. Merge to `main` for production deployment after visual/content sign-off.
