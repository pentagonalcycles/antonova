# TESoul'RA Website

## Local development
- `npm install`
- `npm run dev`

## Quality gates
- `npm run test:unit`
- `npm run test:e2e`
- `npm run build`

## Vercel deployment
1. Import this repository into Vercel.
2. Configure environment variables from `.env.example`.
3. Verify the preview deployment.
4. Promote to production after visual/content sign-off.
